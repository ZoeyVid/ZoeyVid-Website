Mod-Installer

With this mod installer, you can install Iris/Sodium, (Optifine), Quilt, Fabric, NeoForge, Forge and Essential!
And you can also install my Client Modpacks for 1.21.x and also download some texture packs!
In addition, you can save mods in up to 8 profiles and start Minecraft via an extra script and directly select the profile that should be loaded.
A Minecraft Modpack is supported at least until the next version is stable!

You can find the latest download here
Feel free to leave feedback, bugs and suggestions in the issues!

Requirements:

  • Windows 10 or 11 Device with an Internet connection, a Microsoft Account connected to the Microsoft Store and winget
